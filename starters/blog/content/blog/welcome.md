# Welcome to My Blog

This is your first blog post, powered by **Geoff** — a semantically rich static site generator.

## What makes Geoff special?

Every page you write becomes part of a linked data graph. Your content is:

- **Queryable** via SPARQL — build dynamic page listings without a database
- **Validatable** via SHACL — ensure content quality with shape constraints
- **Enriched** with JSON-LD — search engines understand your content better

## Getting started

1. Edit this file in `content/blog/welcome.md`
2. Create new posts with `geoff new blog/my-post.md`
3. Run `geoff serve` to preview with hot reload
4. Run `geoff build` when ready to publish
